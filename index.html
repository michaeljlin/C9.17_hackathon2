Instructions for Quizzical

This is a 2-player trivia game where people take turns answering multiple choice questions. The game works as follows:

  • Player 2 starts by choosing a question for Player 1 to answer
  • After the first question is answered, Player 1 will choose a question for Player 2
  • The game continues to alternate between the players until 30 questions have been attempted
  • The player with the most points is the winner of Quizzical

The questions work as follows:

  • Each question can be drawn from one of 10 different categories.
  • Each category has 3 different difficulties: easy, medium, and hard.
  • Easy questions are worth 10 points, medium questions are 20 points, and hard questions are 40 points.
  • If nothing is chosen for difficulty or category, the default is medium and General Knowledge

There are also hints:

  • On each question, you may use one hint by picking Wikipedia, Youtube, or Twitter
  • Using Wikipedia will reduce the current question value by half.
  • Using Youtube will reduce the current question value by a fourth.
  • Using Twitter will not reduce the question value.
